# 继承自

- [DisplayObject](/zh/docs/api/basic/display-object)

通过 `(x, y)` 定义的位置为圆心。

# 额外属性

### rx

**类型**： `number`

**默认值**：无

**是否必须**：`true`

**说明**：椭圆的水平半径

### ry

**类型**： `number`

**默认值**：无

**是否必须**：`true`

**说明**：椭圆的垂直半径
